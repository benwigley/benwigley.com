Button Select

A jQuery plugin for styling native cross-browser select menus.

Download ZIP

Usage:

$('select.selector').buttonSelect({
  // options go here
});

Options

buttonClass default: 'button'
Default button styling class
class default: ''
Extra custom classes to add to the select button
arrowClass default: ''
Custom class to apply to the arrow icon. This can be useful if you want to use a font-icon as the caret. e.g. 'icon-chevron-down
arrowUpDown default: false
If true, adds two arrow to the select menu (up and down) Note: This does nothing if the arrowClass option is set.

Examples:

Select styling using a custom class

If you use a custom button styling on your site, you can use it for styling select menus too.

$('select.custom').buttonSelect({
  buttonClass: 'button'
});

Select styling using Bootstrap

If you don't use a custom button class, you can always use bootstrap.

$('select').buttonSelect({
  buttonClass: 'btn' // This is the button class that bootstrap uses
});
